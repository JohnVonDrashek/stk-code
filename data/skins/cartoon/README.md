# Cartoon Theme

*Last Updated: 2025-12-30*

Base Cartoon skin theme for SuperTuxKart with a modern, flat UI design.

## Contents

- `stkskin.xml` - Theme configuration and color definitions
- Image assets for UI elements (buttons, dialogs, spinners, checkboxes, etc.)
- `data/` - Additional theme data
- `licenses.txt` - Asset licensing information
