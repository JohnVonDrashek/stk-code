# Online Icons

*Last Updated: 2025-12-30*

Menu icons for online multiplayer features and server interactions.

## Purpose

This directory contains PNG image assets used in the online networking menus. These icons provide visual buttons for finding servers, creating servers, quick play functionality, and online player profiles. Each icon has a normal and hover/focus state variant for UI feedback.

## Contents

| File | Purpose |
|------|---------|
| `menu_create_server.png` | "Create Server" button icon (normal state) |
| `menu_create_server_hover.png` | "Create Server" button icon (hover/focus state) |
| `menu_find_server.png` | "Find Server" button icon (normal state) |
| `menu_find_server_hover.png` | "Find Server" button icon (hover/focus state) |
| `menu_online_player.png` | "Your Profile" button icon (normal state) |
| `menu_online_player_focus.png` | "Your Profile" button icon (hover/focus state) |
| `menu_quick_play.png` | "Quick Play" / "Enter server address" button icon (normal state) |
| `menu_quick_play_hover.png` | "Quick Play" / "Enter server address" button icon (hover/focus state) |
