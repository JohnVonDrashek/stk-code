# Android Icons

*Last Updated: 2025-12-30*

Touch control icons and button graphics for Android and mobile platforms.

## Purpose

This directory contains PNG image assets used for on-screen touch controls on Android and other touch-enabled devices. These icons provide visual feedback for acceleration, steering, drifting, nitro, and other in-game actions when playing with touch input instead of physical controllers.

## Contents

| File | Purpose |
|------|---------|
| `accelerator.png` | Accelerator pedal visual for touch controls |
| `accelerator_handle.png` | Handle/slider element for accelerator control |
| `accelerator_icon.png` | Square icon version of accelerator for settings menus |
| `blur_bg_button.png` | Blurred background for touch buttons (normal state) |
| `blur_bg_button_focus.png` | Blurred background for touch buttons (focused state) |
| `drift.png` | Drift/skid button icon for initiating powerslides |
| `gyroscope_icon.png` | Gyroscope control mode icon for settings and init dialogs |
| `nitro.png` | Nitro boost button icon (filled state) |
| `nitro_empty.png` | Nitro boost button icon (empty/depleted state) |
| `pause.png` | Pause button icon for touch interface |
| `steering_wheel.png` | Virtual steering wheel graphic for steering control mode |
| `steering_wheel_mask_down.png` | Mask overlay for steering wheel (down position) |
| `steering_wheel_mask_up.png` | Mask overlay for steering wheel (up position) |
| `thunderbird_reset.png` | Rescue/reset button icon (uses Thunderbird character) |
| `wing_mirror.png` | Rear-view mirror toggle button icon |
| `licenses.txt` | Copyright and license information for icon assets (CC BY-SA 4.0) |
