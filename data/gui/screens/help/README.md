# Help Screens

*Last Updated: 2025-12-30*

In-game help system with tabbed pages covering gameplay mechanics and features.

## Purpose

This directory contains .stkgui XML files that define the help screen layouts accessible from the main menu. The help system is organized into seven tabbed pages, each covering different aspects of gameplay including basic controls, game modes, powerups, obstacles, story mode progression, kart physics, and multiplayer setup.

## Contents

| File | Purpose |
|------|---------|
| `help1.stkgui` | General tab: Tutorial button, gift boxes, bananas, nitro collection, locked content, skidding mechanics, startup boost tips |
| `help2.stkgui` | Game Modes tab: Regular Race, Time Trial, Follow the Leader, Battle modes (3 Strikes, Free-For-All, CTF), Soccer, Egg Hunt, Ghost Replay, Lap Trial |
| `help3.stkgui` | Powerups tab: BubbleGum, Zipper, Cake, Plunger, Bowling Ball, Parachute, Swapper, Basket Ball, Swatter |
| `help4.stkgui` | Bananas tab: Negative effects from hitting bananas - Anchor, Parachute, Bomb attachments |
| `help5.stkgui` | Story Mode tab: Challenge icons, cup rewards, point system, mystery unlocks for rescuing Gnu from Nolok |
| `help6.stkgui` | Kart Classes tab: Mass differences (light/medium/heavy), acceleration, max speed, nitro efficiency, slipstream mechanics |
| `help7.stkgui` | Multiplayer tab: Online networking (local/global), server creation, splitscreen setup with multiple input devices |
