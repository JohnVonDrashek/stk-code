# Online Screens

*Last Updated: 2025-12-30*

Screen layouts for online multiplayer, user accounts, and networking features.

## Purpose

This directory contains .stkgui XML files that define full-screen layouts for online functionality. These screens handle the main online menu, server browsing and creation, network lobbies, user registration, profile management, and friend lists. They form the core interface for all multiplayer networking features.

## Contents

| File | Purpose |
|------|---------|
| `create_server.stkgui` | Server creation form with name, max players, password, difficulty, and game mode selection |
| `lan.stkgui` | Local networking menu with Find Server and Create Server options for LAN play |
| `network_karts.stkgui` | Kart selection screen for network games with class filter, search, favorites, and timer |
| `networking_lobby.stkgui` | Pre-race lobby with player list, chat interface, emoji support, and race start controls |
| `online.stkgui` | Main online menu with news feed, splitscreen toggle, and buttons for LAN/WAN/profile access |
| `profile_achievements.stkgui` | Simple achievements list view (standalone, no tabs) |
| `profile_achievements_tab.stkgui` | Achievements tab with tabbed navigation (Achievements/Friends/Settings) and player rankings button |
| `profile_friends.stkgui` | Friends tab with friend list and search functionality to find new friends |
| `profile_servers.stkgui` | Global networking menu with Find Server, Create Server, and Quick Play options |
| `profile_settings.stkgui` | Account settings tab with password and email change options |
| `register.stkgui` | User creation screen with tabs for new online account, existing account login, or offline account |
| `server_selection.stkgui` | Server browser with list, search filter, private server toggle, and IPv6 connection option |
| `user_search.stkgui` | Search screen for finding other players by username |
