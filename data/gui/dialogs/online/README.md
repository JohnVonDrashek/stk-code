# Online Dialogs

*Last Updated: 2025-12-30*

Dialog layouts for online multiplayer features, user profiles, and server interactions.

## Purpose

This directory contains .stkgui XML files that define popup dialog layouts for online functionality. These dialogs handle user authentication, account management, server configuration, player interactions, and in-game networking features during online multiplayer sessions.

## Contents

| File | Purpose |
|------|---------|
| `achievement_progress_dialog.stkgui` | Shows achievement goal, description, and progress tree for a selected achievement |
| `change_password.stkgui` | Form for changing online account password with current/new/confirm fields |
| `network_ingame_dialog.stkgui` | In-race pause menu for online games with chat, options to quit server, restart, give up, or return to lobby |
| `player_rankings_dialog.stkgui` | Displays top 10 global player rankings with current player rank and refresh option |
| `recovery_info.stkgui` | Confirmation dialog shown after password recovery request is submitted |
| `recovery_input.stkgui` | Form for account recovery requiring username and email address |
| `registration_terms.stkgui` | Terms and agreement dialog with age verification (13+) for account registration |
| `server_configuration_dialog.stkgui` | Server settings for difficulty, game mode (normal/time trial/battle/soccer), and additional options |
| `server_info_dialog.stkgui` | Server details with player list, password field, and options to join or bookmark |
| `splitscreen_player_dialog.stkgui` | Add/remove splitscreen players for network games with handicap options |
| `user_info_dialog.stkgui` | User profile actions: add/remove friend, accept/decline invite, view profile |
| `vote_dialog.stkgui` | Rating dialog with star-based rating bar for track voting |
